# Çiçi Super AI Chat 🚀

🌟 **MİNİAİ**, Hugging Face API kullanan, web tabanlı bir AI sohbet uygulamasıdır.  
Bu projeyle sen AI ile sohbet edebilir, mini oyunlar oynayabilir ve sesli cevaplar alabilirsin! 😎🎉

---

## 🔹 Özellikler

- 💬 **AI Sohbeti**: Hugging Face distilgpt2 modeli ile gerçek zamanlı cevaplar.  
- 🎮 **Mini Oyunlar**:
  - Taş-Kağıt-Makas 🪨📄✂️
  - 1-100 Sayı Tahmin Oyunu 🔢
- 🗣️ **Sesli Yanıtlar**: AI cevaplarını dinleyebilirsin.  
- 🌗 **Tema Değiştirme**: Karanlık ve açık mod seçenekleri.  
- 💾 **Chat Geçmişi**: LocalStorage sayesinde sayfa yenilense bile sohbetler kaybolmaz.  
- 🎨 **Renkli ve Eğlenceli Arayüz**: Kullanıcı dostu tasarım.
